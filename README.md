# Marquee

Daily movie games. Three games, one app.

## Games
- **Movie Chain** — Connect two actors through shared films
- **The Grid** — Fill a 3×3 grid matching actors to categories  
- **Crosscut** — Swap tiles to find hidden row and column themes

## Project Structure
```
src/
  App.jsx              # Root component, handles screen routing
  MarqueeHome.jsx      # Home screen
  games/
    MovieChain.jsx
    TheGrid.jsx
    Crosscut.jsx
```

## Running Locally
```bash
npm install
npm start
```
